# TLDR

使用以下脚本来一键布署香山开发环境（需要sudo权限）：

This script will setup XiangShan develop environment automatically：

```sh
git clone https://github.com/OpenXiangShan/xs-env
cd xs-env
sudo -s ./setup-tools.sh # apt needs sudo
source setup.sh
```
